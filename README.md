# codingGita-lms-clone
# React + Tailwind Project

## 📌 Project Description

This project is a web application built using **React** and **Tailwind CSS**.
It uses **Vite** as the development build tool for faster development and hot module replacement.

The application runs locally at:

http://localhost:5173/

## ⚙️ Installation

Clone the repository or download the project.

Then install dependencies:

```
npm install
```

---

## ▶️ Run the Project

Start the development server:

```
npm run dev
```

After running the command, open the browser and visit:

```
http://localhost:5173/
```

---

## 🎨 Tailwind CSS Setup

Tailwind CSS is used for styling the project.

Install Tailwind CSS:

```
npm install -D tailwindcss postcss autoprefixer
```

Initialize Tailwind:

```
npx tailwindcss init -p
```

Add Tailwind directives in `index.css`:

```
@tailwind base;
@tailwind components;
@tailwind utilities;
```
